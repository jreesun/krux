## Verify the authenticity
    
* You'll need have [GPG](https://gnupg.org/) installed;
* We recommend installing [GPG4Win](https://www.gpg4win.org/).

----8<----
verify-the-signature-explain.en.txt
----8<----
 
```pwsh
gpg --verify ./{{latest_installer_win}}.sig
```
